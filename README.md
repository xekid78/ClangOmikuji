# ClangOmikuji
おみくじ

## 処理
rand()関数を使用して1 ～ 6のランダムな数字を使って、おみくじの結果を出力します。

## コード
```
#include "stdio.h"

int main(void) {
    printf("hello world\n");
    return 0;
}
```

## 出力結果  
```
hello world
```
